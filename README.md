# Java Laboratory Work #1 

## Опис
Перша лабораторна робота з курсу програмування на Java. Проект охоплює базові аспекти мови: роботу зі змінними, масивами, класами та об'єктами.

##  Технології
**Мова:** Java
**IDE:** IntelliJ IDEA / Eclipse
**Збірка:** Manual compilation

##  Завдання та реалізація

### 1. Арифметичні операції
* Створено 10 цілих змінних.
* Реалізовано операції: `+`, `-`, `*`, `/`.
* Дані виведено в консоль.

### 2. Маніпуляції з рядками
* Кожне слово речення (10-12 слів) збережено в окрему змінну `String`.
* Виведено повне речення одним рядком.

### 3. Об'єктно-орієнтоване моделювання (User)
* Створено клас `User` з персональними даними (id, вік, ім'я, вага, зріст).
* Створено масив з 10 об'єктів.
* Обчислено сумарні показники групи користувачів.

### 4. Робота з класом Car
* Описано клас `Car` з технічними характеристиками (к.с., об'єм двигуна + 5 кастомних полів).
* Реалізовано агрегацію даних для 10 екземплярів.

### 5. Алгоритм реверсу числа
* Використано оператор `%` для розвороту трьохзначного числа (наприклад, 412 -> 214).
