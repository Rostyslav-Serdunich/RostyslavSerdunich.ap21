# Лабораторна робота: 2

Дана програма є консольним застосунком, що об'єднує розв'язання п'яти типових алгоритмічних завдань. Проєкт демонструє навички роботи з розгалуженнями, циклами, математичними функціями та обробкою помилок введення.

---

##  Опис завдань

### 1. Квадратне рівняння
Програма обчислює корені рівняння виду $ax^2 + bx + c = 0$.
* **Логіка**: Обчислюється дискримінант $D = b^2 - 4ac$. Залежно від знаку $D$ визначається кількість коренів (2, 1 або 0).

### 2. Визначення координатного квадранта
Аналіз положення точки на площині за координатами $(x, y)$.
* **Логіка**: Використовуються логічні оператори `&&` (І) для перевірки знаків координат. Якщо одна з координат дорівнює 0, програма повідомляє, що точка лежить на осі.

### 3. Перевірка властивостей цілого числа
Перевірка, чи є введене число одночасно **двозначним** та **парним**.
* **Логіка**: Для коректної роботи з від’ємними числами використовується `Math.abs()`. Умова перевіряє діапазон $[10, 99]$ та залишок від ділення на 2.

### 4. Обчислення кусково-заданої функції
Реалізація обчислення значення $f(x)$ за трьома різними формулами залежно від діапазону $x$:
1.  **$x < -0.5$**: Складна тригонометрична функція.
2.  **$-0.5 \le x \le 1$**: Логарифм за основою 2 (реалізовано через `Math.log(a) / Math.log(2)`).
3.  **$x > 1$**: Корінь з квадрата синуса.

### 5. Робочі та вихідні дні
Визначення типу дня тижня за його порядковим номером (1-7).
* **Логіка**: Використано оператор `switch-case`. Завдяки групуванню кейсів, програма визначає дні 1-5 як робочі, а 6-7 — як вихідні.

###  Обробка виключень (Try-Catch)
Для забезпечення стабільності програми реалізовано метод `readDouble`, який використовує механізм обробки виключень:
* **Блок `try`**: Містить код зчитування `scan.nextDouble()`, який може викликати помилку, якщо користувач введе літери замість цифр.
* **Блок `catch (InputMismatchException)`**: Перехоплює помилку "невідповідності типу". Замість аварійної зупинки, програма виводить попередження та пропонує повторити введення.
* **Очищення буфера**: Команда `scan.next()` всередині `catch` є критично важливою — вона видаляє некоректний символ з пам'яті, щоб запобігти нескінченному циклу помилок.

###  Використані математичні методи класу `Math`
* `Math.sqrt()` — обчислення квадратного кореня.
* `Math.pow(a, b)` — піднесення до степеня.
* `Math.sin()`, `Math.cos()` — тригонометрія.
* `Math.log()` — натуральний логарифм.
* `Math.abs()` — модуль числа для обробки від'ємних значень.
